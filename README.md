# nodejs-web-express-crypto-cookie

## Description
A middleware sets the cookie
name from express default to an
hour limit.

## Tech stack
- nodejs
- npm
  - express
  - cookie-session

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
